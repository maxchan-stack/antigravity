# 批量照片改名工具

一個現代化的網頁介面工具，用於批量重新命名照片檔案。

## 功能特色

- ✨ 現代化深色主題介面
- 📁 掃描資料夾中的 JPG 檔案
- 👀 預覽改名結果
- 🚀 批量執行改名
- 💅 流暢的動畫效果
- 📱 響應式設計

## 使用方法

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

### 2. 啟動應用程式

```bash
python main.py
```

### 3. 開啟瀏覽器

訪問 `http://localhost:8000`

### 4. 使用步驟

1. 輸入資料夾路徑（例如：`/Users/maxchan/Downloads/未命名檔案夾`）
2. 設定檔名前綴（例如：`LINE貼圖`）
3. 設定起始編號和編號位數
4. 點擊「掃描資料夾」查看預覽
5. 確認無誤後點擊「執行改名」

## 檔名格式範例

- 前綴：`LINE貼圖`
- 起始編號：`1`
- 編號位數：`3`

改名結果：
- `LINE貼圖001.jpg`
- `LINE貼圖002.jpg`
- `LINE貼圖003.jpg`
- ...

## 技術架構

- **後端**：FastAPI + Python
- **前端**：Vanilla JavaScript + CSS
- **設計風格**：現代化深色主題

## 注意事項

⚠️ 改名操作無法復原，執行前請確認預覽結果！
