---
description: 自動化整理蝦皮帳務 Excel 並產出 PDF 報表（需詢問月份）
---

此流程會協助使用者整理特定月份的蝦皮帳務資料。

1. **詢問月份範圍**
   - 請先詢問使用者：「請問這次要整理哪幾個月份的資料？（例如：10-12月）」
   - **等待使用者回覆** 後再繼續下一步。

2. **建立/更新處理腳本 (process_orders.py)**
   - 根據使用者指定的月份，動態生成 Python 腳本。
   - 腳本邏輯需包含：
     - 自動搜尋對應月份的 `Order.completed.*.xlsx` 檔案。
     - 密碼 `281105` 解密。
     - 即使某個月份無檔案或無資料，也要強制在報表中列出該月總金額為 0。
     - 產出 `帳務資料_YYYYMMDD.xlsx`（包含當日日期）。

3. **建立/更新 PDF 生成腳本 (create_report_pdf.py)**
   - 設定 PDF 檔名為 `富亭工作室X~Y月銷貨紀錄_YYYYMMDD.pdf`。
   - 讀取輸入檔 `帳務資料_YYYYMMDD.xlsx`。
   - 設定標題資訊：
     - 富亭工作室
     - 統一編號：88356482
     - 銷貨記錄簿
   - 確保表格樣式符合要求（月份總計與總金額需 16級加粗）。

4. **執行腳本**
   - 執行 `python3 process_orders_v4.py` (或最新版本)
   - 執行 `python3 create_report_pdf.py`

5. **完成通知**
   - 告知使用者 Excel 與 PDF 檔案皆已產出，並提供檔案路徑。
