# 教學進度追蹤系統 V10 使用手冊

歡迎使用 **教學進度追蹤系統 V10**。這是一個專為教師設計的輕量級、隱私優先的網頁應用程式，無需安裝任何軟體，直接使用瀏覽器（Chrome, Edge, Safari）即可運作。

## 🌟 特色功能

*   **完全離線運作**：所有資料儲存在您的瀏覽器中，隱私無虞，不會上傳至任何未經授權的伺服器。
*   **響應式設計**：完美支援電腦、平板與手機操作。
*   **多學年管理**：支援多個學期（如 113-1, 113-2）切換，資料獨立管理。
*   **智慧統計**：自動計算段考前、學期結束前的剩餘課堂數，並扣除國定假日與彈性放假。
*   **完整記錄**：輕鬆記錄教學進度、作業、小考安排與課程標籤。
*   **備份與還原**：支援匯出 JSON 格式備份檔，輕鬆將資料轉移至其他裝置。

## 🚀 快速開始

### 1. 啟動系統
只需雙擊打開 `教學進度 V10_乾淨版.html` 檔案即可開始使用。建議將此檔案加入瀏覽器的「我的最愛」以便快速存取。

### 2. 初始設定
第一次使用時，請點擊右上角的 **「設定」** (⚙️ 圖示) 進行基礎配置：

1.  **建立班級**：
    *   進入「排課」分頁。
    *   輸入班級名稱（如：七年一班、803）。
    *   **重要**：勾選該班級每週的上課日（星期一至五）。系統需要此資訊來計算剩餘堂數與顯示月曆。
    *   點擊「+」新增。

2.  **設定學期與段考日期**：
    *   進入「段考」分頁。
    *   設定 **學期起訖日期**（如：開學日與結業式日期）。這對於計算剩餘堂數至關重要。
    *   （選填）設定第一次、第二次段考與期末考日期。系統會自動顯示距離下次考試的剩餘堂數。

3.  **檢查假期**：
    *   進入「假期」分頁。系統預設已載入部分國定假日。
    *   若學校有特殊的彈性放假或校慶補假，請在此手動新增，或在「月曆模式」中直接點擊日期切換。

## 📖 功能操作說明

### 📝 新增進度 (首頁)
在首頁的「新增進度」卡片中：
1.  **選擇班級**：從下拉選單選擇。
2.  **填寫內容**：
    *   **科目**：輸入教學科目。
    *   **日期**：預設為今天，可手動修改補登。
    *   **進度/章節**：輸入今日教學內容（系統會智慧記憶並推薦最近的進度）。
    *   **標籤**：點選標籤（如 #實驗, #複習），方便日後篩選。
3.  **儲存**：點擊儲存按鈕。

> **💡 小技巧**：如果今天有多個班級上同一進度，填寫完一筆後，可以直接切換「班級」但保留「進度內容」繼續新增，省時又快速！

### 📊 查看紀錄
系統提供兩種檢視模式，可從右上角或底部導航列切換：

1.  **列表模式 (List)**：
    *   依班級分類顯示所有歷史紀錄。
    *   點擊班級卡片可展開/收合詳細紀錄。
    *   支援關鍵字搜尋（可搜尋進度、日期或標籤）。
    *   每筆紀錄右側提供 **編輯** (✏️) 與 **刪除** (🗑️) 功能。

2.  **月曆模式 (Calendar)**：
    *   以月曆形式總覽課程分布。
    *   **實心顏色**：代表當天已上過課。
    *   **虛線框**：代表該班級當天有排課但尚未填寫進度（提醒您補登）。
    *   **點擊日期**：可切換該日為「彈性假期」（橘色標示）。

### ☁️ 備份與雲端 (設定 > 系統)

*   **資料備份**：
    *   點擊「下載備份檔」，系統會生成一個 `.json` 檔案。請妥善保存此檔案。
    *   若要更換電腦，在新電腦上開啟網頁，點擊「讀取備份檔 (還原)」並選擇剛才的 `.json` 檔案即可。

*   **雲端同步 (進階功能)**：
    *   若您有架設 Google Apps Script (GAS)，可在此填入 URL 以啟用雲端同步功能，將資料備份至 Google試算表。

## ❓ 常見問題 (FAQ)

**Q: 我的資料會不見嗎？**
A: 資料儲存在您瀏覽器的 `localStorage` 中。只要不清除瀏覽器快取 (Cache/Cookies)，資料都會保留。但強烈建議您**定期下載備份檔**，以免電腦重灌或瀏覽器故障導致資料遺失。

**Q: 可以用手機開啟嗎？**
A: 可以！將 HTML 檔案傳送到手機，用瀏覽器開啟即可。iOS 用戶可使用「加入主畫面」功能，體驗如同原生 App。

**Q: 為什麼剩餘堂數計算不準確？**
A: 請檢查「設定 > 段考」中的 **學期起訖日期** 是否正確。系統只會計算在學期範圍內的上課日。

**Q: 如何列印報表？**
A: 在列表模式下，使用瀏覽器的列印功能 (Ctrl+P / Cmd+P)。系統已內建列印專用樣式，會自動隱藏不必要的按鈕，呈現乾淨的報表格式。
