# 專案資料夾說明

本專案已分為兩個獨立版本，請根據需求選擇對應的資料夾進行部署：

## 1. 校園通用版 (V9.0 Campus Edition)
📂 **資料夾**：`gas_project_v9_campus`

- **用途**：提供給其他老師或學校使用。
- **特色**：
    - 移除所有個人資訊 (Email 等)。
    - 移除特定學科 (物理) 與品牌名稱。
    - 標題為「校園段考成績查詢」。
    - 適合公開分享或建立範本。

## 2. 個人專用版 (V9.0 Physics Edition)
📂 **資料夾**：`gas_project_personal`

- **用途**：MaxChan 個人物理科使用。
- **特色**：
    - 保留個人 Email (`maxgdodo@gmail.com`)。
    - 保留「物理科」標題與 "PHYSICS DEPARTMENT" 副標題。
    - 保留星宇航空主題色註解。
    - 包含 V9.0 所有最新功能修正 (防護機制、趨勢顏色等)。
